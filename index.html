Instructions on how to play

The Learning the Lingo game is simple to play.

Firstly add your name and age when requested and select the Play Game button.

In the next page you will be given a page that displays a question, 3 possible answers and your score. Select the answer you think is correct and a message will appear letting you know if you are correct or not. If you are correct, select the Next Question button and the next question will be displayed with 3 possible answers. If you got the question wrong, select the OK button on the message that appears and try answering the question again. There is no limit to how many attempts you have on answering the question. This game is about learning!

Your score will update as you move through the questions.

Once you complete the 10 questions, you will see your score on the Leaderboard on the main page.

Enjoy playing our little language learning game.
